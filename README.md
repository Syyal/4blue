# 4blue

video para processo seletivo 4blue

Esta automação foi desenvolvida utilizando o n8n para processar dados de novos alunos recebidos via Webhook, realizar o tratamento e padronização das informações, armazenar os dados em um banco PostgreSQL e enviar um e-mail de confirmação da matrícula.

O fluxo garante a integridade dos dados e automatiza o processo de matrícula, reduzindo erros manuais e melhorando a comunicação com os alunos.
